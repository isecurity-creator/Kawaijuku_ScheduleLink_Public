## 導入方法
- 本ページのファイルを全てダウンロードしてください。「Chrome-Extention」フォルダは、ファイル構造が崩れるとエラーの原因となるため、Zip形式でのダウンロード(この場合展開が必要です)、もしくはGit Cloneを推奨します。
- [GoogleAppsScript](https://script.google.com/home)にアクセスし、「新しいプロジェクト」をクリックしてください。
- 先ほどダウンロードしたファイルのうち「GCalender_AutoAdd.gs」の中身を全てコピーして、AppsScript側に貼り付けてください。
- Googleカレンダーを開き、必要に応じて新しいカレンダーを作成してください。(AppsScriptと同じGoogleアカウントでログインしているか、編集権限があるカレンダーが必要です)
- 「カレンダーの設定」→「カレンダーの統合」と進み、表示されるカレンダーIDをコピーし、AppsScriptの3行目にある「カレンダーIDをここに指定」部分に貼り付けてください。
- AppsSCript画面右上の「デプロイ」→「新しいデプロイ」と進み、「次のユーザーとしてデプロイ」を自身のGoogleアカウント、「アクセスできるユーザー」を全員にしてデプロイしてください。
- 権限の許可を求められた場合は、画面に従って許可してください。
- 「ウェブアプリ」のURLをコピーしてください。
- 先ほどダウンロードしたファイルのうち「Chrome-Extention」内にある「data_get_autoyear.js」を開き、4行目の「ここにデプロイしたURLを指定」部分に貼り付けてください。
- GoogleChromeまたはMicrosoftEdgeで[拡張機能のページ](chrome://extensions/)にアクセスし、「開発者モード」のトグルをONにします。
- 「展開して読み込み」をクリックし、先ほどダウンロードした「Chrome-Extention」フォルダを指定してください。
- これで導入は完了です。
