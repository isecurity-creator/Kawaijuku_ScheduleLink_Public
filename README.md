# Kawaijuku_ScheduleLink_Public

## この拡張機能について
- 河合塾の塾生マイページに表示される予定表を、授業実施教室も含め、Googleカレンダー側に自動で転記する拡張機能です。
- 詳細は[河合wikiのページ](https://p2kawaijuku.wiki.fc2.com/wiki/%E5%A1%BE%E7%94%9F%E3%83%9E%E3%82%A4%E3%83%9A%E3%83%BC%E3%82%B8)をご確認ください。

## 導入方法
- 本ページのファイルを全てダウンロードしてください。「Chrome-Extention」フォルダは、ファイル構造が崩れるとエラーの原因となるため、Zip形式でのダウンロード(この場合展開が必要です)、もしくはGit Cloneを推奨します。
- [GoogleAppsScript](https://script.google.com/home)にアクセスし、「新しいプロジェクト」をクリックしてください。
- 先ほどダウンロードしたファイルのうち「GCalender_AutoAdd.gs」の中身を全てコピーして、AppsScript側に貼り付けてください。
- Googleカレンダーを開き、必要に応じて新しいカレンダーを作成してください。(AppsScriptと同じGoogleアカウントでログインしているか、編集権限があるカレンダーが必要です)
- 「カレンダーの設定」→「カレンダーの統合」と進み、表示されるカレンダーIDをコピーし、AppsScriptの3行目にある「カレンダーIDをここに指定」部分に貼り付けてください。
- AppsSCript画面右上の「デプロイ」→「新しいデプロイ」と進み、「次のユーザーとしてデプロイ」を自身のGoogleアカウント、「アクセスできるユーザー」を全員にしてデプロイしてください。
- 権限の許可を求められた場合は、画面に従って許可してください。
- 「ウェブアプリ」のURLをコピーしてください。
- 先ほどダウンロードしたファイルのうち「Chrome-Extention」内にある「data_get_autoyear.js」を開き、4行目の「ここにデプロイしたURLを指定」部分に貼り付けてください。
- GoogleChromeまたはMicrosoftEdgeで拡張機能のページ(chrome://extensions/)にアクセスし、「開発者モード」のトグルをONにします。
- 「展開して読み込み」をクリックし、先ほどダウンロードした「Chrome-Extention」フォルダを指定してください。
- 「Kawaijuku_roomID_get」という拡張機能が読み込まれたことを確認してください。不具合防止のため、この時点ではトグルをOFFにしておいてください。
- これで導入は完了です。

## 使用方法
- 塾生マイページを開いてログインし、「予定」タブを開きます。
- 「前週」をクリックし、カレンダーが表示されたら、拡張機能のページ(chrome://extensions/)を別タブで開き「Kawaijuku_roomID_get」のトグルをONにしてください。
- 塾生マイページのタブに戻り「翌週」ボタンを押してください。
- 表示されているカレンダーの年月を自動取得し、Alertで表示されますので、取得されている年月とカレンダーの年月が一致していることを確認してください。正しくない場合や、「自動取得できませんでした」と出た場合は、手動で入力してください。
- 「データの送信を開始してもよろしいでしょうか？」という確認が出るので、問題なければOKをクリックしてください。
- 「送信完了しました。」と出れば送信が完了しています。Googleカレンダー側に反映されていることを確認してください。なお、同じ名前・同じ日時の予定がある場合は上書きされるので、重複して登録されることは基本的にないはずです。
- 「翌週」ボタンを押していき、操作を繰り返すことで、各週におけるスケジュールをGoogleカレンダー側に自動登録することができます。
- 作業が終わったら、拡張機能のページ(chrome://extensions/)を別タブで開き「Kawaijuku_roomID_get」のトグルをOFFにしてください。

## 動作環境
- この拡張機能の使用には、GoogleChromeまたはMicrosoftEdgeの最新版がインストールされた、WindowsまたはMacが必要です。
- スマートフォンやタブレットでは動作しません。
